# Проект project_4

## Описание проекта:

Проект project_4 - создан как учебный, для работы HTML-кодом с помощью Bootstrap ,для создания проекта в Django.

##  Установка

1. Клонируйте репозиторий:
```
git@github.com:olgakostenko240/project_4.git
```
2. Установите зависимости:
```
poetry install
```
3. Установите пакет для работы с Django
```
poetry add django
```

## Использование:

1. Создано новое приложение catalog
2. В приложении catalog создана Главная страница в файле home.html.
3. В приложении catalog создана страница Контакты в файле contacts.html.
4. Создан файл с фикстурами catalog_fixture.json для приложения catalog.
5. Создана кастомная команда для наполнения базы даных.
6. Создан базовый шаблон base.html
7. Создана директория includes с подшаблонами
8. Создана директория templatetags с функцией для медиа
9. Создано новое приложение blogs.
